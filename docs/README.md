# /docs 
> 기존의 단순 `Odometry` 기반 위치 추정에서 벗어나, 보다 안정적이고 정확한 로컬라이제이션을 구현하기 위한 알고리즘을 학습하고 적용하기 위한 디렉토리입니다.

### **1. INN (Intensity-based Nearest Neighbor)**
- LiDAR intensity 정보를 기반으로 주변 지형과 매칭하여 위치를 추정하는 방식으로, 주행 환경 변화에 강한 특성을 활용할 예정이다.

### 2. NDT (Normal Distributions Transform)
- LiDAR 스캔을 확률 분포로 변환하여 맵과 정합하는 방식으로, 정밀한 포즈 보정이 가능해 실제 주행 안정성을 높일 수 있다.

### 3. EKF-SLAM (Extended Kalman Filter SLAM)
- 센서 예측값과 실제 관측값을 결합해 로봇의 위치와 환경 지도를 동시에 추정하는 알고리즘으로, 맵 기반 자율주행의 핵심 기술 중 하나다.

### 4. 센서 퓨전 (LiDAR + IMU + Encoder)
- LiDAR: 절대적인 주변 구조물 기반 위치 보정
- IMU: 순간적인 회전·가속 변화 보강
- Encoder: 장기적인 이동 거리 추적
  <br>
  
  이 세 센서의 장점을 조합하여, 주행 중 발생하는 잡음·드리프트·슬립 같은 문제를 최소화하고 안정적인 포즈 추정을 달성하는 것을 목표로 한다.
