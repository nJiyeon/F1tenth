## 1. EKF-SLAM 기본 개념

EKF-SLAM(Extended Kalman Filter SLAM)는 **로봇의 위치(state)** 와 **주변 랜드마크(map)** 를 동시에 추정하는 알고리즘이다.

> **핵심:** 로봇 위치 + 환경의 랜드마크 위치를 하나의 큰 상태 벡터로 묶고, EKF로 반복적으로 업데이트한다.

## 상태 벡터
```
X = [로봇 pose (x, y, θ), 랜드마크1(x1, y1), 랜드마크2(x2, y2), ...]
```

## EKF 동작 과정

### 1) 예측(Prediction Step)
- 로봇의 motion model을 이용해 다음 위치를 예측한다.
- 예측 과정에서 불확실성(공분산)이 증가한다.

### 2) 갱신(Update Step)
- LiDAR/Camera 등으로 랜드마크를 관측한다.
- 관측 모델을 기반으로 상태와 공분산을 보정한다.
- 기존 랜드마크와의 상관관계를 활용하여 로봇 위치와 맵을 정교하게 추정한다.

> 예측 → 관측 → 보정의 반복을 통해 점점 정밀한 맵과 pose를 생성한다.

---

## 2. EKF-SLAM의 장점

- 랜드마크–로봇 간 **상관관계를 유지**하여 서로 보정하는 구조이다.
- 센서 노이즈에 강하며 로봇 움직임이 불안정해도 비교적 안정적인 추정이 가능하다.
- 중간 규모(moderate scale) 환경에서 robust하게 동작한다.
