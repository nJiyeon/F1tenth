## 1. 실습 목적
- SLAM을 활용해 실제 주행 환경의 지도를 직접 구축함으로써, 자율주행 알고리즘 개발을 위한 기반 데이터를 확보한다.
- 구축된 맵과 waypoint를 이용해 실제 주행 없이도 시뮬레이션 상에서 자율주행 알고리즘을 반복적으로 검증·개선할 수 있는 환경을 조성한다.
- 대회장 및 확장 맵과 동일한 조건의 가상 주행 환경을 마련하여, 이후 알고리즘 수정 및 안정성 향상을 위한 테스트를 쉽게 수행하도록 한다.
    <br><br>
    
## 2. 실습 과제
- SLAM으로 알파시티 맵 따기
: 로봇으로 직접 SLAM을 돌려서 실제 주행에 사용할 맵(.pgm, .yaml)을 만든다.
- Waypoint 기록해서 주행 경로 만들기'
: 만든 맵 위에서 waypoint logger를 사용해 주행할 경로 데이터를 수집한다.
- 자율주행 테스트 환경 세팅하기
: 맵 + waypoint 기반으로 시뮬 환경과 실제 주행 환경을 설정하고, 이후 알고리즘 수정·테스트가 가능하도록 준비한다.
  <br><br>
